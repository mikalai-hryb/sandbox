------ Дано ------

- Консольное приложение для работы с НБРБ. 
- Описание сущностей и API находятся на сайте НБРБ (http://www.nbrb.by/APIHelp/ExRates).
- Для REST запросов можно воспользоваться классом HttpClient.
- Для сериализации можно воспользоваться библиотекой Newtonsoft.

------ Описание задания ------

1. Приложение должно иметь возможность считывания значений сигнатур валют из файла:
	- Сигнатуры будут перечислены в файле через запятую, например : USD, EUR, RUB;
	- Приложение выводит в результатирующий файл информацию по данным валютам;
	- По каждой валюте нужно вывести максимально подробную информацию;
	- Блоки информации должны быть упорядоченны по именам сигнатур;

2. Приложение должно иметь возможность ввода сигнатур самим пользователем:
	- Пользователь имеет возможность ввести в консоль название сигнатур;
	- В консоли отображается результат по сигнатурам;
	- Выводимые записи должны быть упорядочены по убыванию по Cur_ID;

3. Приложение должно давать возможность просмотреть валюту с максимальным/минимальным рейтом:
	- Выводится максимальная информация по максимальной/минимальной валюте;

4. В систему доджна быть встроена минимальная система логирования:
	- Логирование совершается в файл с произвольным именем;
	- Файл логирования должен быть визуально понятен на уровне простого пользователя;
	- Достаточно логирования только начала/конца определённых процесов;

------ Что в итоге-то ------

Получится консольное приложение с примитивным GUI, где пользователь может посмотреть курсы валют. 
Для работы с коллекциями использовать LINQ синтаксис, любой удобный для Вас.
В целом в конце у Вас сложится базовое понятие работы с файлами и LINQ синтаксисом.
